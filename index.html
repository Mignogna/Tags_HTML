Tags HTML

Existem centenas de tags HTML, cada uma com uma finalidade específica na criação e estruturação de uma página web. Para facilitar o entendimento, podemos agrupá-las em algumas categorias principais:

Tags Estruturais Básicas

Essas tags são o alicerce de qualquer documento HTML, definindo a estrutura fundamental da página.

  • <!DOCTYPE html>: Embora não seja uma tag, é a declaração de tipo de documento. Ela informa ao navegador qual versão do HTML está sendo usada (neste caso, HTML5). É crucial para o navegador renderizar a página corretamente.
  • <html>: A tag raiz que engloba todo o conteúdo da página. Tudo o que você vê no navegador está dentro desta tag.
  • <head>: Contém metadados sobre a página, como o título que aparece na aba do navegador, links para folhas de estilo (CSS) e scripts JavaScript, e informações de codificação de caracteres. O conteúdo do <head> não é visível diretamente na página.
  • <body>: Contém todo o conteúdo visível da página, como texto, imagens, vídeos, links e formulários.

Tags de Conteúdo de Texto

Essas tags são usadas para formatar e organizar o texto dentro do <body>.

  • Títulos: <h1> a <h6> - Usadas para definir títulos e subtítulos. <h1> é o mais importante e <h1> o menos. Elas ajudam a estruturar o conteúdo e são importantes para SEO (otimização para mecanismos de busca).

    Este é um <h6>
  • Parágrafos: <p> - Define um parágrafo de texto. É a tag mais comum para blocos de texto.
  • Ênfase e Destaque:
    <strong>: Indica texto com forte importância, geralmente renderizado em negrito. Ex: Hello World!
    <em>: Indica ênfase, geralmente renderizado em itálico. Ex: Hello World!
    <b>: Usado para negrito sem importância semântica. Ex: Hello World!
    <i>: Usado para itálico sem importância semântica (por exemplo, termos técnicos ou frases em outro idioma). Ex: Hello World!
    <mark>: Usado para realçar partes de um texto. Ex: Hello World!
    <sub>: Usado para subescrever o texto. Ex: Hello World28!
    <sup>: Usado para superescrever o texto. Ex: Hello World28!
  • Quebra de Linha e Linha Horizontal:
    <br>: Insere uma quebra de linha.
    <hr>: Cria uma linha horizontal, geralmente para separar seções de conteúdo.
    (Esta é a linha↓)
  • Listas:
    <ul>: Cria uma lista não ordenada (com marcadores).
    <ol>: Cria uma lista ordenada (com números ou letras).
    <li>: Define um item de lista dentro de <ul> ou <ol>.

Tags de Links e Mídia

Essas tags permitem adicionar links e incorporar mídia à sua página.

  • Links: <a> - Usada para criar hiperlinks. O atributo href especifica o URL de destino.
    Ex: Turismo em São Paulo
  • Imagens: <img> - Incorpora uma imagem na página. O atributo src especifica o caminho da imagem e alt fornece texto alternativo para acessibilidade.
  • Vídeos: <video> - Incorpora um vídeo na página.
  • Áudios: <audio> - Incorpora um arquivo de áudio na página.

Tags de Formulários

Essas tags são usadas para criar formulários interativos que permitem aos usuários inserir dados.

  • <form>: Define um formulário HTML.
  • <label>: Associa um rótulo a um controle de formulário.
  • <input>: Cria vários tipos de campos de entrada (texto, senha, botão de rádio, checkbox, etc.), dependendo do atributo type.
  • <textarea>: Cria uma área de texto de várias linhas.
  • <button>: Cria um botão clicável.
  • <select>: Cria uma lista suspensa de opções.
  • <option>: Define uma opção dentro de um <select>.

Tags Semânticas e Estruturais (HTML5)

Com o HTML5, surgiram tags que ajudam a dar mais significado à estrutura da página, melhorando a acessibilidade e o SEO.

  • <header>: Representa o cabeçalho de uma seção ou de toda a página (geralmente contém logotipos, títulos, navegação).
  • <nav>: Contém links de navegação.
  • <main>: Representa o conteúdo principal do<body>.
  • <article>: Representa um conteúdo independente e auto-contido (como um post de blog ou notícia).
  • <section>: Agrupa conteúdo relacionado.
  • <aside>: Representa conteúdo que está relacionado à seção principal, mas é separado (como uma barra lateral ou informações adicionais).
  • <footer>: Representa o rodapé de uma seção ou de toda a página (geralmente contém informações de direitos autorais, links de contato).
  • <div>: Uma tag genérica de bloco, muito usada para agrupar elementos para fins de estilo com CSS. Não tem significado semântico por si só.
  • <span>: Uma tag genérica de linha, usada para aplicar estilos a pequenas porções de texto. Também não tem significado semântico.

Tags de Estilo e Script

Essas tags são usadas para incorporar CSS e JavaScript na sua página.

  • <style>: Usada para escrever estilos CSS diretamente no documento HTML (dentro do <head>).
  • <link>: Usada para vincular folhas de estilo externas (CSS). É a forma preferida de adicionar CSS.
  • <script>: Usada para incorporar código JavaScript ou para vincular arquivos JavaScript externos.


Entender as diferentes tags HTML e como usá-las corretamente é o primeiro passo para construir qualquer página web. A prática e a exploração de novas tags são essenciais para dominar a linguagem.
